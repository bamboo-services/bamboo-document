# ContextHolder

`ContextHolder` 基于 `ThreadLocal` 实现请求级别的上下文管理。每个 HTTP 请求在进入过滤器时初始化上下文，在请求结束时清理，确保线程安全且无上下文泄漏。

## 内部结构

`ContextHolder` 内部使用 `RequestContext` 记录类存储上下文数据：

```java title="ContextHolder.java"
// [!code highlight:2]
// 私有记录类，封装上下文标识与起始时间
private record RequestContext(String contextId, long startTime) {}
```

## 生命周期

```
请求进入 → ContextFilter → initContext() → 业务处理 → ResultUtil（读取上下文）→ clear()
```

1. **初始化**：`ContextFilter` 在请求进入时调用 `initContext()` 生成上下文
2. **使用**：业务代码和 `ResultUtil` 通过静态方法读取上下文信息
3. **清理**：`ContextFilter` 在请求结束后调用 `clear()` 释放 ThreadLocal 资源

```java title="ContextHolder.java"

// 清理 ThreadLocal，防止内存泄漏

```

## 使用示例

### 手动管理上下文

```java title="ManualContextExample.java"
// [!code highlight:2]
// 初始化上下文（通常由 ContextFilter 自动完成）
ContextHolder.initContext();

try {
    // [!code highlight:2]
    // 获取上下文标识
    String contextId = ContextHolder.getContextId();
    log.info("当前请求上下文: {}", contextId);

    // 执行业务逻辑...

    // [!code highlight:2]
    // 获取请求耗时
    Long duration = ContextHolder.getDuration();
    log.info("请求处理耗时: {}ms", duration);
} finally {
    // [!code highlight:2]
    // 必须在 finally 块中清理，确保资源释放
    ContextHolder.clear();
}
```

### 使用外部传入的上下文标识

```java title="ExternalContextExample.java"
String externalUuid = request.getHeader("X-Context-UUID");
if (externalUuid != null) {
    // [!code highlight:2]
    // 使用请求头中的 UUID 初始化上下文
    ContextHolder.initContext(externalUuid);
} else {
    ContextHolder.initContext();
}
```

### 条件判断

```java title="ConditionalExample.java"
// [!code highlight:2]
// 在非 HTTP 请求场景中安全使用
if (ContextHolder.hasContext()) {
    String contextId = ContextHolder.getContextId();
    log.info("请求 {} 处理中...", contextId);
} else {
    log.info("非 HTTP 请求环境，跳过上下文追踪");
}
```

## 注意事项

- **线程安全**：`ContextHolder` 基于 `ThreadLocal` 实现，每个线程拥有独立的上下文副本
- **资源清理**：必须在请求结束时调用 `clear()` 方法。`ContextFilter` 已在内部处理此逻辑，无需手动调用
- **异步场景**：若业务代码使用 `@Async` 或线程池，上下文不会自动传播至新线程。需在新线程中手动初始化
- **与 ResultUtil 的协作**：`ResultUtil` 内部自动调用 `ContextHolder` 的方法注入响应字段，无需额外操作
- **空值场景**：`getContextId/getStartTime/getDuration` 在未初始化上下文时均可能返回 `null`
