# Spring MVC 模块

`bamboo-mvc` 是筱工具(Java) 面向 Spring MVC 应用的集成模块，提供开箱即用的异常处理、请求过滤、日志切面与上下文管理能力。该模块通过 Spring Boot 自动配置机制完成所有组件的注册，无需手动声明 Bean。

## 安装

在 `pom.xml` 中添加依赖：

```xml title="pom.xml"
<dependency>
    <groupId>com.x-lf.utility</groupId>
    <artifactId>bamboo-mvc</artifactId>
    <version>2.0.0-beta2</version>
</dependency>
```

> `bamboo-mvc` 会自动引入 `bamboo-base` 核心库，无需重复声明。

## 核心组件

| 组件类别 | 说明 | 核心类 |
|---------|------|--------|
| **响应工具** | 标准化 HTTP 响应构建 | `ResultUtil` |
| **上下文管理** | 基于 ThreadLocal 的请求上下文 | `ContextHolder` |
| **异常处理** | 分层异常捕获与标准化响应 | `SystemExceptionHandler` 等 |
| **AOP 切面** | 自动化日志记录与调试控制 | `LogAspectHandler` 等 |
| **过滤器** | 请求预处理与安全控制 | `ContextFilter` 等 |
| **工具类** | HTTP 请求头与处理器解析 | `HeaderUtil`、`HttpServletUtil` |

## 自动配置

`BaseSdkAutoConfiguration` 通过 `@Configuration` 和 `@ComponentScan` 自动扫描并注册所有组件：

```java title="BaseSdkAutoConfiguration.java"
// [!code highlight:3]
@Configuration
@EnableAspectJAutoProxy
@ComponentScan
public class BaseSdkAutoConfiguration {
    // 自动注册所有过滤器、异常处理器、AOP 切面
}
```

自动配置完成以下注册：

- `ContextFilter` — 请求上下文初始化过滤器（最高优先级）
- `LogAspectHandler` — Controller / Service / Repository 日志切面
- `DebugAspectHandler` — 调试模式控制切面
- 全部异常处理器链（Java 基础 → 公共业务 → Spring MVC → 数据库）
- `MybatisPlusConfiguration` — MyBatis-Plus 分页插件（按需激活）

## 配置属性

```yaml title="application.yml"
bamboo:
  context:
    enable-input: true
    exclude-urls:
      - /actuator/**
```

## 模块架构

```
bamboo-mvc
├── ResultUtil                   # 标准化响应构建
├── ContextHolder                # ThreadLocal 上下文管理
├── exception/                   # 异常处理器层级
│   ├── JavaBaseExceptionHandler     # Java 基础异常
│   ├── PublicExceptionHandler       # 公共业务异常
│   ├── SystemExceptionHandler       # Spring MVC 异常
│   ├── MysqlExceptionHandler        # MySQL 异常
│   ├── PostgreSqlExceptionHandler   # PostgreSQL 异常
│   └── UserAuthExceptionHandler     # 用户认证异常
├── aspect/                      # AOP 切面
│   ├── LogAspectHandler             # 日志切面
│   ├── DebugAspectHandler           # 调试切面
│   └── BusinessLogAspect            # 业务日志
├── filter/                      # 请求过滤器
│   ├── ContextFilter                # 上下文注入
│   ├── AllowCorsFilterHandler       # CORS 跨域
│   ├── AllowOptionFilterHandler     # OPTIONS 预检
│   ├── BanEmptyReferer              # Referer 校验
│   └── PermissionFilterHandler      # 权限校验（抽象）
└── util/                        # 工具类
    ├── HeaderUtil                   # 请求头解析
    └── HttpServletUtil              # Servlet 工具
```
