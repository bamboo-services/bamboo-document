# SnowflakeUtil

`SnowflakeUtil` 是雪花算法的静态工具类，位于 `com.xlf.utility.utility` 包下。封装了 `SnowflakeIdGenerator` 的能力，提供简洁的静态方法用于 ID 生成、解析与验证。

## 初始化

`SnowflakeUtil` 在使用前必须完成初始化。在 Spring Boot 环境中，框架通过 `SnowflakeUtilInitializer` 自动完成；在非 Spring 环境中，需手动调用：

```java title="Example.java"
import com.xlf.utility.utility.SnowflakeUtil;
import com.xlf.utility.incrementer.SnowflakeIdGenerator;

// [!code highlight:3]
// 创建生成器并初始化工具类
SnowflakeIdGenerator generator = new SnowflakeIdGenerator(1, 1, 1690214400000L);
SnowflakeUtil.initialize(generator);
```

## 使用示例

### ID 生成

```java title="UserService.java"
import com.xlf.utility.utility.SnowflakeUtil;

public class UserService {

    public Long createUser(UserDTO dto) {
        // [!code highlight:2]
        // 生成用户唯一 ID
        long userId = SnowflakeUtil.generate();
        // 保存用户...
        return userId;
    }
}
```

### 基因模式生成

```java title="OrderService.java"
import com.xlf.utility.utility.SnowflakeUtil;

public class OrderService {

    public Long createOrder(Long userId) {
        // [!code highlight:3]
        // 使用用户 ID 的哈希值作为基因，确保同一用户的订单路由至同一分片
        int gene = (int) (userId % 8);
        long orderId = SnowflakeUtil.generateWithGene(gene);
        return orderId;
    }
}
```

### ID 解析

```java title="Example.java"
import com.xlf.utility.utility.SnowflakeUtil;
import com.xlf.utility.models.dto.SnowflakeInfoDTO;

long id = 123456789012345678L;

// [!code highlight:2]
// 完整解析
SnowflakeInfoDTO info = SnowflakeUtil.parse(id);
System.out.println("生成时间: " + info.getGeneratedAt());
System.out.println("数据中心: " + info.getDatacenterId());
System.out.println("机器 ID: " + info.getMachineId());
System.out.println("序列号: " + info.getSequence());

// [!code highlight:2]
// 单独提取时间戳
long timestamp = SnowflakeUtil.parseTimestamp(id);
```

### ID 验证

```java title="Example.java"
import com.xlf.utility.utility.SnowflakeUtil;

// [!code highlight:2]
// 验证 ID 合法性
boolean valid = SnowflakeUtil.isValid(123456789012345678L);
if (!valid) {
    throw new BusinessException(ErrorCode.PARAMETER_INVALID, "无效的 ID 格式");
}
```
