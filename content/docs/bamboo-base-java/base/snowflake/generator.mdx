# SnowflakeIdGenerator

`SnowflakeIdGenerator` 是雪花算法的核心实现类，位于 `com.xlf.utility.incrementer` 包下。该类通过位运算与同步机制，保证在分布式环境中生成全局唯一、趋势递增的 64 位长整型 ID。

    // [!code highlight:4]

        // 参数校验与初始化
    }

    public synchronized long nextId() { ... }
}

## 位布局常量

生成器内部定义了以下位布局常量，决定 64 位 ID 中各段的位置与容量：

| 常量 | 值 | 说明 |
|------|------|------|
| `TIMESTAMP_BITS` | 41 | 时间戳占用位数 |
| `DATACENTER_BITS` | 3 | 数据中心 ID 占用位数 |
| `MACHINE_BITS` | 4 | 机器 ID 占用位数 |
| `GENE_BITS` | 3 | 基因位占用位数 |
| `SEQUENCE_BITS` | 12 | 序列号占用位数 |
| `MAX_DATACENTER_ID` | 7 | 数据中心 ID 最大值 (2^3 - 1) |
| `MAX_MACHINE_ID` | 15 | 机器 ID 最大值 (2^4 - 1) |
| `MAX_SEQUENCE` | 4095 | 序列号最大值 (2^12 - 1) |

## ID 生成流程

`nextId()` 方法使用 `synchronized` 关键字保证线程安全，其核心流程如下：

```java title="SnowflakeIdGenerator.java"
public synchronized long nextId() {
    long currentTimestamp = System.currentTimeMillis();

    // [!code highlight:4]
    // 时钟回拨检测
    if (currentTimestamp < lastTimestamp) {
        throw new RuntimeException("时钟回拨，拒绝生成 ID");
    }

    // [!code highlight:5]
    // 同一毫秒内序列号自增
    if (currentTimestamp == lastTimestamp) {
        sequence = (sequence + 1) & MAX_SEQUENCE;
        if (sequence == 0) {
            // 序列号溢出，等待下一毫秒
            currentTimestamp = waitNextMillis(lastTimestamp);
        }
    } else {
        sequence = 0;
    }

    lastTimestamp = currentTimestamp;

    // [!code highlight:6]
    // 位运算组装 64 位 ID
    return ((currentTimestamp - epoch) << (DATACENTER_BITS + MACHINE_BITS + GENE_BITS + SEQUENCE_BITS))
         | (datacenterId << (MACHINE_BITS + GENE_BITS + SEQUENCE_BITS))
         | (machineId << (GENE_BITS + SEQUENCE_BITS))
         | (gene << SEQUENCE_BITS)
         | sequence;
}
```

### 流程说明

1. **获取当前时间戳** - 取系统当前毫秒时间
2. **时钟回拨检测** - 若当前时间早于上次生成时间，抛出异常拒绝生成，防止 ID 重复
3. **序列号管理** - 同一毫秒内序列号自增；若溢出（超过 4095），阻塞等待下一毫秒
4. **位运算组装** - 将时间戳、数据中心 ID、机器 ID、基因位、序列号按位拼接为 64 位长整型

## 时钟回拨处理

当检测到系统时钟回拨时，生成器将直接抛出 `RuntimeException`。这是一种保守策略，确保在任何情况下都不会生成重复 ID。在生产环境中，建议通过 NTP 服务保证时钟同步。

## 使用示例

通常不需要直接使用 `SnowflakeIdGenerator`，而是通过 `SnowflakeUtil` 的静态方法调用。若需手动创建生成器：

```java title="Example.java"
import com.xlf.utility.incrementer.SnowflakeIdGenerator;

// [!code highlight:2]
// 手动创建生成器实例
SnowflakeIdGenerator generator = new SnowflakeIdGenerator(1, 1, 1690214400000L);

// [!code highlight:2]
// 生成 ID
long id = generator.nextId();
```
