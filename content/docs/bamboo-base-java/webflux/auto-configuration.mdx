# 自动配置

`bamboo-webflux` 通过 Spring Boot 自动配置机制，在应用启动时自动注册所有基础设施组件。开发者只需引入 Maven 依赖，无需手动编写 `@Bean` 配置。

## WebFluxSdkAutoConfiguration

`WebFluxSdkAutoConfiguration` 是 `bamboo-webflux` 的自动配置入口类，通过条件注解确保仅在响应式 Web 应用中生效。

```java title="WebFluxSdkAutoConfiguration.java"
// [!code highlight:3]
// 仅当应用类型为 REACTIVE 时激活
@Configuration
@ConditionalOnWebApplication(type = ConditionalOnWebApplication.Type.REACTIVE)
public class WebFluxSdkAutoConfiguration {
    // ...
}
```

### 自动注册的组件

| 组件 | 类型 | 说明 |
|------|------|------|
| `contextFilter` | WebFilter | 上下文注入过滤器，注入链路追踪 UUID 与开始时间 |
| `logAspectHandler` | Aspect | 日志切面，记录请求/响应日志 |
| `debugAspectHandler` | Aspect | 调试切面，控制调试接口的访问权限 |
| `errorController` | Controller | 错误控制器 |
| `mybatisPlusConfigHandler` | Configuration | MyBatis-Plus 配置（条件注册） |

### 条件注册

`mybatisPlusConfigHandler` 仅在类路径中存在 MyBatis-Plus 相关类时才会注册：

```java title="WebFluxSdkAutoConfiguration.java"
// [!code highlight:3]
// 仅当 MyBatis-Plus 存在于类路径时注册
@Bean
@ConditionalOnClass(name = "com.baomidou.mybatisplus.core.MybatisPlusAutoConfiguration")
public MybatisPlusConfigHandler mybatisPlusConfigHandler() {
    return new MybatisPlusConfigHandler();
}
```

## ContextWebFluxProperties

`ContextWebFluxProperties` 是 `bamboo-webflux` 的配置属性类，绑定 `utility.context.webflux` 前缀。

```java title="ContextWebFluxProperties.java"
// [!code highlight:2]
// 配置前缀为 utility.context.webflux
@ConfigurationProperties("utility.context.webflux")
public class ContextWebFluxProperties {
    private boolean enableInput = true;
    private boolean enableOutput = true;
    private boolean gatewayMode = false;
    private List<String> excludeUrls;
}
```

### 属性说明

      default: '空列表',
    },

### 配置示例

```yaml title="application.yml"
utility:
  context:
    webflux:
      enable-input: true
      enable-output: true
      gateway-mode: false
      exclude-urls:
        - /actuator/**
        - /health
        - /favicon.ico
```

## 与 MVC 自动配置的对比

`bamboo-mvc` 与 `bamboo-webflux` 的自动配置互斥，Spring Boot 会根据应用类型自动选择：

| 条件 | 激活的配置 |
|------|-----------|
| `@ConditionalOnWebApplication(Type.SERVLET)` | `MvcSdkAutoConfiguration` |
| `@ConditionalOnWebApplication(Type.REACTIVE)` | `WebFluxSdkAutoConfiguration` |

因此在同一个 Spring Boot 应用中，不应同时引入 `bamboo-mvc` 与 `bamboo-webflux` 依赖。
