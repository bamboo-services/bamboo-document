# LogAspectHandler

`LogAspectHandler` 是 `bamboo-webflux` 模块的日志切面处理器，通过 AOP 自动记录 Controller 层的请求与响应日志。与 MVC 版本的核心差异在于，该切面能够正确处理 `Mono` 与 `Flux` 响应式返回类型。

## 类定义

```java title="LogAspectHandler.java"
// [!code highlight:3]
// 响应式日志切面，由自动配置注册
@Aspect
public class LogAspectHandler {
    // ...
}
```

## 切点定义

`LogAspectHandler` 拦截所有 `@RestController` 注解标注的 Controller 类中的方法：

```java title="LogAspectHandler.java"
// [!code highlight:2]
// 匹配所有 @RestController 标注类的所有方法
@Around("@within(org.springframework.web.bind.annotation.RestController)")
public Object around(ProceedingJoinPoint joinPoint) throws Throwable {
    // ...
}
```

## 响应式返回类型处理

在 Spring WebFlux 中，Controller 方法通常返回 `Mono` 或 `Flux` 类型。日志切面需要在响应式管道完成后才能获取到实际的响应数据，因此处理逻辑与同步版本有本质区别。

### Mono 返回类型

```java title="LogAspectHandler.java"
// [!code highlight:6]
// 对于 Mono 返回类型，使用 doOnSuccess / doOnError 记录日志
if (result instanceof Mono<?> mono) {
    return mono
            .doOnSuccess(value -> logResponse(joinPoint, value))
            .doOnError(error -> logError(joinPoint, error));
}
```

### Flux 返回类型

```java title="LogAspectHandler.java"
// [!code highlight:5]
// 对于 Flux 返回类型，使用 doOnComplete / doOnError 记录日志
if (result instanceof Flux<?> flux) {
    return flux
            .doOnComplete(() -> logComplete(joinPoint))
            .doOnError(error -> logError(joinPoint, error));
}
```

### 非响应式返回类型

对于直接返回非响应式对象的方法，切面按照同步方式处理日志记录。

## ServerWebExchange 提取

在 WebFlux 环境中，切面从方法参数中提取 `ServerWebExchange` 以获取请求信息：

```java title="LogAspectHandler.java"
// [!code highlight:5]
// 遍历方法参数，查找 ServerWebExchange 实例
ServerWebExchange exchange = null;
for (Object arg : joinPoint.getArgs()) {
    if (arg instanceof ServerWebExchange) {
        exchange = (ServerWebExchange) arg;
        break;
    }
}
```

通过 `ServerWebExchange` 可获取请求路径、请求方法、客户端 IP 等信息用于日志记录。

## 日志输出内容

日志切面记录的信息包括：

| 字段 | 说明 |
|------|------|
| 请求路径 | HTTP 请求的 URI |
| 请求方法 | GET、POST、PUT、DELETE 等 |
| 客户端 IP | 从 `ServerWebExchange` 中提取 |
| 方法签名 | Controller 类名与方法名 |
| 执行耗时 | 方法执行时间（毫秒） |
| 响应状态 | 成功或异常 |

## 自动注册

`LogAspectHandler` 由 `WebFluxSdkAutoConfiguration` 自动注册，无需手动配置：

```java title="WebFluxSdkAutoConfiguration.java"
@Bean
public LogAspectHandler logAspectHandler() {
    return new LogAspectHandler();
}
```
