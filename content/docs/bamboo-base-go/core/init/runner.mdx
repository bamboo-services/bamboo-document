# 服务运行时

`xMain.Runner` 用于统一启动 HTTP 服务、托管后台协程并在信号到达时优雅关闭。

```go
import xMain "github.com/bamboo-services/bamboo-base-go/major/main"
```

## Runner

```go title="runner.go"
func Runner(
    reg *xReg.Reg,
    log *xLog.LogNamedLogger,
    opts []xOption.Option,
    goroutineFunc ...func(ctx context.Context, extra ...any),
)
```

## 功能

- 从环境变量读取 `XLF_HOST` 与 `XLF_PORT` 作为监听地址，默认 `localhost:1118`。
- 支持传入 `*xOption.Config`，自动装配缓存、数据库等内置组件
- 自动监听 `SIGINT`/`SIGTERM`，触发优雅关闭。
- 支持注册路由函数（传统方式）或通过 Option 声明式注册路由
- 支持启动多个后台协程并统一等待退出。
- HTTP 关闭超时时间为 30 秒。

## 快速使用

```go title="main.go"
package main

import (
    "context"

    xCtx "github.com/bamboo-services/bamboo-base-go/defined/context"
    xLog "github.com/bamboo-services/bamboo-base-go/common/log"
    xMain "github.com/bamboo-services/bamboo-base-go/major/main"
    xOption "github.com/bamboo-services/bamboo-base-go/major/option"
    xReg "github.com/bamboo-services/bamboo-base-go/major/register"
    xRegNode "github.com/bamboo-services/bamboo-base-go/major/register/node"
)

func main() {
    reg := xReg.Register(context.Background(), []xRegNode.RegNodeList{
        {Key: xCtx.DatabaseKey, Node: initDatabase},
        {Key: xCtx.RedisClientKey, Node: initRedis},
    })

    log := xLog.WithName(xLog.NamedMAIN)

    // 通过 Option 声明式配置路由
    opts := []xOption.Option{
        xOption.WithRoute(registerRoute),
    }

    xMain.Runner(reg, log, opts)
}
```

## 仅路由（无内置组件）

```go
opts := []xOption.Option{
    xOption.WithRoute(registerRoute),
}
xMain.Runner(reg, log, opts)
```

## 带后台任务示例

```go
func runConsumer(ctx context.Context, extra ...any) {
    for {
        select {
        case <-ctx.Done():
            return
        default:
            // 消费任务
        }
    }
}

func main() {
    // ... reg, log 初始化省略
    opts := []xOption.Option{
        xOption.WithRoute(registerRoute),
    }
    xMain.Runner(reg, log, opts, runConsumer)
}
```

> 如需接入 gRPC（可选），请前往独立模块文档：[`gRPC 运行时`](/docs/bamboo-base-go/grpc/runner)

## Option 声明式配置

v1.0.1 起 Runner 接受 `[]xOption.Option`，以声明式装配内置组件：

```go title="main.go"
import (
    xLog "github.com/bamboo-services/bamboo-base-go/common/log"
    xMain "github.com/bamboo-services/bamboo-base-go/major/main"
    xOption "github.com/bamboo-services/bamboo-base-go/major/option"
    xReg "github.com/bamboo-services/bamboo-base-go/major/register"
)

func main() {
    reg := xReg.Register(context.Background(), nil)
    log := xLog.WithName(xLog.NamedMAIN)

    // 声明式配置
    opts := []xOption.Option{
        xOption.WithRedis("localhost:6379"),
        xOption.WithMySQL("user:pass@tcp(localhost:3306)/bamboo"),
        xOption.WithRoute(registerRoute),
    }

    // Runner 根据 opts 自动装配 Redis 客户端、GORM DB 等
    xMain.Runner(reg, log, opts)
}
```

> nil 或空 `[]xOption.Option` 表示不启用任何内置实现，仅以 reg 现有状态启动。
