## 关于

竹简文档是 Bamboo 服务生态的统一文档平台，旨在为开发者提供清晰、完整的组件库使用指南。

### 特点

- **多语言支持** - 覆盖 Go、Java、TypeScript 等主流语言
- **统一规范** - 各语言库遵循相似的设计理念和 API 风格
- **持续更新** - 文档与代码同步维护
