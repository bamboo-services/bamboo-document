# 入门指南

## 安装

Bamboo Base 需要 Go 1.24.6 或更高版本。

```bash
go get github.com/bamboo-services/bamboo-base-go
```

## 最小可用示例

创建一个简单的 HTTP 服务：

```go
package main

import (
    "github.com/bamboo-services/bamboo-base-go/web"
    "github.com/bamboo-services/bamboo-base-go/web/result"
    "github.com/gin-gonic/gin"
)

func main() {
    // 使用默认配置创建引擎
    r := web.Default()

    // 注册路由
    r.GET("/ping", func(c *gin.Context) {
        result.Success(c, "pong")
    })

    // 启动服务
    r.Run(":8080")
}
```

## 项目初始化

推荐使用 Bamboo Base 的注册系统进行项目初始化：

```go
package main

import (
    "github.com/bamboo-services/bamboo-base-go/web"
    "github.com/bamboo-services/bamboo-base-go/web/result"
    "github.com/gin-gonic/gin"
)

func main() {
    // 初始化 Reg 实例
    web.InitRegister()

    // 初始化配置
    web.Register.ConfigInit()

    // 初始化日志
    web.Register.LoggerInit()

    // 初始化雪花算法
    web.Register.SnowflakeInit(1, 1)

    // 初始化 Gin 引擎
    web.Register.EngineInit()

    // 初始化系统上下文
    web.Register.SystemContextInit()

    // 注册路由
    web.Register.Engine.GET("/api/hello", func(c *gin.Context) {
        result.SuccessHasData(c, "Hello, Bamboo Base!")
    })

    // 启动服务
    web.Register.Run(":8080")
}
```

## 环境配置

创建 `.env` 文件：

```bash
# 服务配置
SERVER_PORT=8080
SERVER_MODE=debug

# 日志配置
LOG_LEVEL=info
LOG_PATH=./logs

# 雪花算法配置
SNOWFLAKE_DATACENTER_ID=1
SNOWFLAKE_NODE_ID=1
```
